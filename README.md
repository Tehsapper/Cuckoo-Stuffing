Cuckoo stuffing
===
Java implementation of [Cuckoo byte stuffing algorithm][1].

Maven dependency
---
```xml
<dependency>
  <groupId>com.github.sirikid</groupId>
  <artifactId>cuckoo-stuffing</artifactId>
  <version>X.Y.Z</version>
</dependency>
```
where `X.Y.Z` release number, e. g. [latest release][2]
